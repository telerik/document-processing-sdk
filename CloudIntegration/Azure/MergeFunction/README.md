# Azure PDF Merge Solution

A serverless solution for merging PDF documents using Azure Functions and Telerik Document Processing.

## Included Projects

- **MergeFunctionApp** - Azure Function that performs PDF merging
- **MergeApp** - Console test client

## Dependencies

- [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)
- [Azure Functions Core Tools v4](https://docs.microsoft.com/azure/azure-functions/functions-run-local)
- [Telerik Document Processing](https://docs.telerik.com/devtools/document-processing/) - Check the [Pricing and Licensing FAQ](https://www.telerik.com/faqs/pricing-and-licensing)
- [Azure Functions Documentation](https://docs.microsoft.com/azure/azure-functions/)
